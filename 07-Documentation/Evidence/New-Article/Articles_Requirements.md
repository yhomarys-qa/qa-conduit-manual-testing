# Articles Requirements

## Title
- Required field
- Must be 2–40 characters
- Must allow letters, numbers, special characters
- Must not allow duplicate titles

## Description
- Required field
- Must validate length constraints
- Must not contain invalid formatting (extra spaces rules)

## Body
- Required field
- Must allow text input only
- Must not allow attachments
- Must support multi-line editing

## Tags
- Optional field
- Must start with #
- Must allow multiple tags
- Must not allow duplicates
- Must be case-insensitive
