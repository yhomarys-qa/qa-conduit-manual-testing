# Home Page Requirements

## Feed Behavior
- Guest users see Global Feed
- Logged users see "Your Feed"
- Articles must be ordered by most recent

## Pagination
- Maximum 10 articles per page
- Navigation between pages must work correctly

## Article Interaction
- User can like/unlike articles
- User cannot like same article multiple times

## Navigation
- User can open article by title, description, or read more link
- User can access author profile via username
