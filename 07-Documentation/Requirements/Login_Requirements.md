# Login Requirements

## Authentication Methods
- The system must allow login using Email or Username
- Email input must be case-insensitive
- Username must be accepted when valid

## Input Validation
- Email must follow valid format rules
- Password field is required
- System must reject invalid credentials

## UI Behavior
- The "Email or Username" field must accept both formats
- Placeholder must correctly indicate expected input
- "Forgot your password?" link must be visible

## Error Handling
- Invalid credentials must show consistent error message
- System must not allow authenticated users to access login page
