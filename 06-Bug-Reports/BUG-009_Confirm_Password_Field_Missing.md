# 🐞 BUG-009 - Confirm Password Field Is Not Displayed in Registration Form

## 📌 Summary
The registration form does not display the required "Confirm Password" field, which is essential for validating password confirmation during user registration.

---

## 🧪 Preconditions
- Registration page is accessible in the browser  
- User is not authenticated  

---

## 🔁 Steps to Reproduce
1. Access the registration page  
2. Observe the fields displayed in the registration form  

---

## ❌ Actual Result
The "Confirm Password" field is not displayed in the registration form.

---

## ✅ Expected Result
The system should display the "Confirm Password" field below the "Password" field to ensure proper password confirmation validation.

---

## 📊 Severity
High

---

## 📎 Environment
- Windows 10  
- Chrome 145.0.7632.76 (64-bit)  
- App version: Demo  

---
