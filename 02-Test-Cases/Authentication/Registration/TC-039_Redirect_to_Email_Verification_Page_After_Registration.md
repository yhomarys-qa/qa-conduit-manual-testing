# TC-039 - Redirect to Email Verification Page After Registration

## Steps
1. Open the Conduit application
2. Navigate to the Sign up page
3. Enter valid registration data (Username, Email, Password, Confirm Password)
4. Click on the "Sign up" button

## Expected Result
After successful registration, the user should be automatically redirected to the email verification page.
The page should prompt the user to enter the 6-digit verification code sent to their email address.
