# TC-015 - Username Invalid Characters Not Allowed

## Steps
1. Open the Conduit application
2. Navigate to the Sign up page
3. Enter a Username containing invalid characters (e.g., "user@123", "user#name", "user name")
4. Fill in Email, Password, and Confirm Password with valid data
5. Click on the "Sign up" button

## Expected Result
The system should display a validation message indicating that the Username contains invalid characters.
The user should not be registered.
