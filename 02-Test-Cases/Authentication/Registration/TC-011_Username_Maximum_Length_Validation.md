# TC-011 - Username Maximum Length Validation

## Steps
1. Open the Conduit application
2. Navigate to the Sign up page
3. Enter a Username with more than 40 characters
4. Fill in Email, Password, and Confirm Password with valid data
5. Click on the "Sign up" button

## Expected Result
The system should display a validation message indicating that the Username must not exceed 40 characters.
The user should not be registered.
