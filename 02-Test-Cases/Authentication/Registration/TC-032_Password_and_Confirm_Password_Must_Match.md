# TC-032 - Password and Confirm Password Must Match

## Steps
1. Open the Conduit application
2. Navigate to the Sign up page
3. Enter a valid Username
4. Enter a valid Email
5. Enter a valid Password (e.g., "Senha123@")
6. Enter a different value in Confirm Password (e.g., "Senha123")
7. Click on the "Sign up" button

## Expected Result
The system should display a validation message indicating that Password and Confirm Password do not match.
The user should not be registered.
