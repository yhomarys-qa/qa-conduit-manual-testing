# TC-037 - Successful Registration With Valid Data

## Steps
1. Open the Conduit application
2. Navigate to the Sign up page
3. Enter a valid Username
4. Enter a valid Email
5. Enter a valid Password
6. Enter the same value in Confirm Password
7. Click on the "Sign up" button

## Expected Result
The user should be registered successfully.
A confirmation email should be sent to the registered email address.
The user should be redirected to the email verification page (or next registration step).
