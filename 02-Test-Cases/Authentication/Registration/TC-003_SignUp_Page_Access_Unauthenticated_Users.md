# TC-003 - Sign Up Page Access for Unauthenticated Users

## Steps
1. Ensure the user is not authenticated
2. Open the Conduit application
3. Navigate to the "Sign up" page via the navigation bar or direct URL
4. Observe the page access behavior

## Expected Result
Unauthenticated users should be able to access the Sign up page successfully.
The registration form should be displayed without any restrictions.
