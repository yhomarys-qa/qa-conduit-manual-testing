# TC-027 - Password Must Contain Uppercase Letter

## Steps
1. Open the Conduit application
2. Navigate to the Sign up page
3. Enter a valid Username
4. Enter a valid Email
5. Enter a Password without uppercase letters (e.g., "senha123@")
6. Enter the same value in Confirm Password
7. Click on the "Sign up" button

## Expected Result
The system should display a validation message indicating that the Password must contain at least one uppercase letter.
The user should not be registered.
