# TC-014 - Username Case Insensitive Validation

## Steps
1. Open the Conduit application
2. Navigate to the "Register" page
3. Enter a username with different uppercase and lowercase letters (for example, "TestUser", "testuser" or "TESTUSER")
4. Enter valid values ​​for the Email, Password, and Confirm Password fields
5. Click the "Register" button

## Expected Result
The system should treat usernames without case sensitivity.
