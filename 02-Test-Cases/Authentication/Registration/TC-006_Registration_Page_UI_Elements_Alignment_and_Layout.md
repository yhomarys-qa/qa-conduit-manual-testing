# TC-006 - Registration Page UI Elements Alignment and Layout

## Steps
1. Open the Conduit application
2. Navigate to the "Sign up" page
3. Observe the registration form layout
4. Check the alignment of input fields (Username, Email, Password, Confirm Password)
5. Verify the alignment of labels, placeholders, and the "Sign up" button

## Expected Result
All UI elements on the Registration page should be properly aligned and visually consistent.
Input fields, labels, and buttons should be displayed in a clear and structured layout without overlap or misalignment.
