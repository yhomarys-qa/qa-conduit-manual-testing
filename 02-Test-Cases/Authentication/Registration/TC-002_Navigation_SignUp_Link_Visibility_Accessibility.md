# TC-002 - Navigation Sign Up Link Visibility and Accessibility

## Steps
1. Open the Conduit application
2. Ensure the user is not logged in
3. Observe the navigation bar
4. Check if the "Sign up" link is visible and clickable

## Expected Result
The "Sign up" link should be visible in the navigation bar for unauthenticated users.
The link should be clickable and redirect the user to the Registration page.
