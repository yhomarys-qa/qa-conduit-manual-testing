# TC-036 - Sign Up Using Enter Key

## Steps
1. Open the Conduit application
2. Navigate to the Sign up page
3. Enter a valid Username
4. Enter a valid Email
5. Enter a valid Password
6. Enter the same value in Confirm Password
7. Press the Enter key on the keyboard

## Expected Result
The registration form should be submitted successfully when the Enter key is pressed.
The user should be registered successfully.
