# TC-042 - “Already Have an Account?” Link Visibility on Sign Up Page

## Steps
1. Open the Conduit app.
2. Navigate to the Registration page.
3. Open the registration form and check if the "Already have an account?" link is displayed.

## Expected Result
The "Already have an account?" link should be visible and correctly displayed on the Sign up page.
The user should be able to identify the link without any interaction.
