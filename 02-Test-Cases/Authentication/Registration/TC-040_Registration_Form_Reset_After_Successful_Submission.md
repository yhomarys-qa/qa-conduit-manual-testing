# TC-040 - Registration Form Reset After Successful Submission

## Steps
1. Open the Conduit application
2. Navigate to the Sign up page
3. Enter valid registration data (Username, Email, Password, Confirm Password)
4. Click on the "Sign up" button
5. After successful registration, return to the Sign up page (if accessible again)

## Expected Result
After a successful registration, the form fields should be reset when the user returns to the Sign up page.
No previously entered data should remain in any input field.
