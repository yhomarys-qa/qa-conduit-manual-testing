# TC-039 - Email Verification Code Entry

## Steps
1. Open the Conduit application
2. Complete the registration process successfully
3. Navigate to the email verification page
4. Enter the 6-digit verification code received 
5. Submit the verification code

## Expected Result
The system should accept a valid 6-digit verification code.
The user should be successfully verified and allowed to proceed to the next step (e.g., login or home page).
