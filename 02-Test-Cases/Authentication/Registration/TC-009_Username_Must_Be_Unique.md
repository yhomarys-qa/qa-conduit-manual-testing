# TC-009 - Username Must Be Unique

## Steps
1. Open the Conduit application
2. Navigate to the Sign up page
3. Register a new user with valid Username, Email, Password, and Confirm Password
4. Complete the registration successfully
5. Navigate again to the Sign up page
6. Attempt to register using the same Username with a different Email
7. Click on the "Sign up" button

## Expected Result
The system should display a validation message indicating that the Username is already in use.
The user should not be able to create a duplicate account with the same Username.
