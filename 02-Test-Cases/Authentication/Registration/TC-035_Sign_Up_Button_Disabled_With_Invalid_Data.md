# TC-035 - Sign Up Button Disabled With Invalid Data

## Steps
1. Open the Conduit application
2. Navigate to the Sign up page
3. Leave one or more required fields empty (Username, Email, Password, or Confirm Password)
4. Or enter invalid data in any field (e.g., invalid email format or weak password)
5. Observe the state of the "Sign up" button

## Expected Result
The "Sign up" button should remain disabled when required fields are empty or contain invalid data.
The user should not be able to submit the registration form.
