# TC-034 - Sign Up Button Enabled With Valid Data

## Steps
1. Open the Conduit application
2. Navigate to the Sign up page
3. Enter a valid Username
4. Enter a valid Email
5. Enter a valid Password that meets all requirements
6. Enter the same value in Confirm Password
7. Observe the state of the "Sign up" button

## Expected Result
The "Sign up" button should be enabled when all required fields contain valid data.
The user should be able to click the button to submit the registration form.
