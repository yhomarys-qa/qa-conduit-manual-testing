# TC-024 - Password Field Is Required

## Steps
1. Open the Conduit application
2. Navigate to the Sign up page
3. Enter a valid Username
4. Enter a valid Email
5. Leave the Password field empty
6. Enter a valid value in Confirm Password
7. Click on the "Sign up" button

## Expected Result
The system should display a validation message indicating that the Password field is required.
The user should not be registered.
