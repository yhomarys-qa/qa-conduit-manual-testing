# TC-025 - Password Minimum Length Validation

## Steps
1. Open the Conduit application
2. Navigate to the Sign up page
3. Enter a valid Username
4. Enter a valid Email
5. Enter a Password with less than 8 characters (e.g., "Ab1@12")
6. Enter the same value in Confirm Password
7. Click on the "Sign up" button

## Expected Result
The system should display a validation message indicating that the Password must be at least 8 characters long.
The user should not be registered.
