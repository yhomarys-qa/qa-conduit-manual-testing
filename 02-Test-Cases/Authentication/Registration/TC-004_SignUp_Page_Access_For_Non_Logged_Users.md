# TC-004 - Sign Up Page Access For Non Logged Users

## Steps
1. Ensure the user is not logged in
2. Open the Conduit application
3. Navigate to the "Sign up" page via the navigation bar or direct URL
4. Observe the page access behavior

## Expected Result
Non-authenticated users should be able to access the Registration (Sign Up) page successfully.
The registration form should be displayed without restrictions.
