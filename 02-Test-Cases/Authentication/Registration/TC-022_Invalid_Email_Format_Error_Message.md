# TC-022 - Invalid Email Format Error Message

## Steps
1. Open the Conduit application
2. Navigate to the Sign up page
3. Enter a valid Username
4. Enter an invalid Email format (e.g., "useremail.com" or "user@com")
5. Fill in Password and Confirm Password with valid data
6. Click on the "Sign up" button

## Expected Result
The system should display an error message indicating that the Email format is invalid (e.g., "This email looks invalid.").
The user should not be registered.
