# TC-021 - Email Allowed Special Characters

## Steps
1. Open the Conduit application
2. Navigate to the Sign up page
3. Enter a valid Username
4. Enter an Email containing valid special characters in the local part (e.g., "user.name+test-email_1@example.com")
5. Fill in Password and Confirm Password with valid data
6. Click on the "Sign up" button

## Expected Result
The system should accept valid Email formats that include allowed special characters (+, -, _, .) in the local part.
The user should be registered successfully if all other fields are valid.
