# TC-012 - Username Must Start With a Letter

## Steps
1. Open the Conduit application
2. Navigate to the Sign up page
3. Enter a Username that starts with a number or special character (e.g., "1user" or "_user")
4. Fill in Email, Password, and Confirm Password with valid data
5. Click on the "Sign up" button

## Expected Result
The system should display a validation message indicating that the Username must start with a letter.
The user should not be registered.
