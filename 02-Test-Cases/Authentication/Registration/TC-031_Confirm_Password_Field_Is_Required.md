# TC-031 - Confirm Password Field Is Required

## Steps
1. Open the Conduit application
2. Navigate to the Sign up page
3. Enter a valid Username
4. Enter a valid Email
5. Enter a valid Password
6. Leave the Confirm Password field empty
7. Click on the "Sign up" button

## Expected Result
The system should display a validation message indicating that the Confirm Password field is required.
The user should not be registered.
