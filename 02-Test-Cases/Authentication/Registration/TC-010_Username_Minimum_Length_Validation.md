# TC-010 - Username Minimum Length Validation

## Steps
1. Open the Conduit application
2. Navigate to the Sign up page
3. Enter a Username with less than 3 characters (e.g., "ab")
4. Fill in Email, Password, and Confirm Password with valid data
5. Click on the "Sign up" button

## Expected Result
The system should display a validation message indicating that the Username must be at least 3 characters long.
The user should not be registered.
