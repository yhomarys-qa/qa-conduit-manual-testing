# TC-016 - Username Whitespace Not Allowed

## Steps
1. Open the Conduit application
2. Navigate to the Sign up page
3. Enter a Username containing only whitespace or spaces (e.g., " " or "name user")
4. Fill in Email, Password, and Confirm Password with valid data
5. Click on the "Sign up" button

## Expected Result
The system should display a validation message indicating that the Username cannot contain spaces or be empty.
The user should not be registered.
