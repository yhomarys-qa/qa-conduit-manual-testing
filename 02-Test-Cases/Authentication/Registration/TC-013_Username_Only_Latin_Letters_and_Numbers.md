# TC-013 - Username Accepts Only Latin Letters and Numbers

## Steps
1. Open the Conduit application
2. Navigate to the Sign up page
3. Enter a Username containing non-Latin characters or unsupported symbols (e.g., "usuário", "用户123", "user@name")
4. Fill in Email, Password, and Confirm Password with valid data
5. Click on the "Sign up" button

## Expected Result
The system should display a validation message indicating that the Username must contain only Latin letters and numbers.
The user should not be registered.
