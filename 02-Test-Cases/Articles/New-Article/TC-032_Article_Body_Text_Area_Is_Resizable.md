# TC-032 - Article Body Text Area Should Be Resizable

## Steps
1. Open the Conduit application
2. Log in with valid credentials
3. Navigate to "New Article"
4. Locate the bottom-right corner of the "Write your article" text area
5. Drag the corner horizontally and vertically to resize the field

## Expected Result
The text area should resize appropriately.
The page layout should remain stable.
No user interface elements should oerlap.
