# TC-004 - Duplicate Article Title Warning Message

## Steps
1. Open the Conduit application
2. Log in with valid credentials
3. Navigate to "New Article"
4. Enter a title that already exists
5. Fill all other required fields with valid data
6. Click on "Publish Article"

## Expected Result
The system should display the warning message:
"Article with a similar Title already exists. Do you want to continue?"
