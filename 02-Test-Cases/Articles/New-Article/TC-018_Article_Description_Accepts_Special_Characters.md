# TC-018 - Article Description Accepts Special Characters

## Steps
1. Open the Conduit application
2. Log in with valid credentials
3. Navigate to "New Article"
4. Enter a description using special characters (e.g., "!@#$%")
5. Fill all other required fields with valid data
6. Click on "Publish Article"

## Expected Result
The system should accept the description and be created successfully.
