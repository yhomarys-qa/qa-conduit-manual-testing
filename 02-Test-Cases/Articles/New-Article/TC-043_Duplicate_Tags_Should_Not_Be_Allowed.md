# TC-043 - Duplicate Tags Should Not Be Allowed

## Steps
1. Open the Conduit application
2. Log in with valid credentials
3. Navigate to "New Article"
4. Add the same tag twice (e.g., "#test", "#test")
5. Fill all other required fields with valid data
6. Click on "Publish Article"

## Expected Result
The system should prevent duplicate tags; only one instance of the tag should be saved.
