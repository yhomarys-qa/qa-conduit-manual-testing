# TC-037 - Tag Must Start with "#"

## Steps
1. Open the Conduit application
2. Log in with valid credentials
3. Navigate to "New Article"
4. Enter a tag without the "#" symbol (e.g., "testtag")
5. Fill all other required fields with valid data
6. Click on "Publish Article"

## Expected Result
The system should display a validation error for invalid tag format.
