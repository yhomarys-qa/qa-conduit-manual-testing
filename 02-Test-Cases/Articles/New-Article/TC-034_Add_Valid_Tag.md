# TC-034 - Add Valid Tag to Article

## Steps
1. Open the Conduit application
2. Log in with valid credentials
3. Navigate to "New Article"
4. Enter a valid tag in the "Enter tags" field (e.g., "#test")
5. Fill all other required fields with valid data
6. Click on "Publish Article"

## Expected Result
The system should accept the tag.
The article should be successfully created.
