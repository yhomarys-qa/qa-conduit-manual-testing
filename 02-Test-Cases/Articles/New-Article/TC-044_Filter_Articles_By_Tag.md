# TC-044 - Articles Can Be Filtered by Tag

## Steps
1. Open the Conduit application
2. Log in with valid credentials
3. Create a new article with a unique tag (e.g., "#testing")
4. Ensure the article is successfully published
5. Navigate to the home page
6. Locate the "Popular Tags" section
7. Click on the created tag

## Expected Result
The system should display only articles associated with the selected tag.
