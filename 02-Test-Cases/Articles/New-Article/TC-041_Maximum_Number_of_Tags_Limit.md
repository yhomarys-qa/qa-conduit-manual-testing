# TC-041 - Maximum Number of Tags Validation

## Steps
1. Open the Conduit application
2. Log in with valid credentials
3. Navigate to "New Article"
4. Add 11 valid tags (e.g., "#tag1", "#tag2", ..., "#tag11")
5. Fill all other required fields with valid data
6. Click on "Publish Article"

## Expected Result
The system should prevent the article from being created with more than 10 tags.
