# TC-042 - User Can Create Article with Tags Successfully

## Steps
1. Open the Conduit application
2. Log in with valid credentials
3. Navigate to "New Article"
4. Enter valid title, description, and body
5. Add valid tags (e.g., "#technology", "#testing")
6. Click on "Publish Article"
7. Navigate to the created article page

## Expected Result
The article should be successfully created.
The tags should be displayed correctly on the article page.
