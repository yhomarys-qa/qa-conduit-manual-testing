# TC-030 - Article Body Should Not Contain Multiple Consecutive Spaces

## Steps
1. Open the Conduit application
2. Log in with valid credentials
3. Navigate to "New Article"
4. Enter a body with multiple consecutive spaces (e.g., "Body test  content with  multiple  spaces inside")
5. Fill all other required fields with valid data
6. Click on "Publish Article"

## Expected Result
The system should display a validation error message.
