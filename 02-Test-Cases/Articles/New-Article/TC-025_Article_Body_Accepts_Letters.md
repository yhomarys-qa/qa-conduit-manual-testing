# TC-025 - Article Body Accepts Letters

## Steps
1. Open the Conduit application
2. Log in with valid credentials
3. Navigate to "New Article"
4. Enter an article body using letters (e.g., "This is a test article body content")
5. Fill all other required fields with valid data
6. Click on "Publish Article"

## Expected Result
The system should accept the article body and create it successfully.
