# TC-031 - Article Body Should Not Accept File Attachments

## Steps
1. Open the Conduit application
2. Log in with valid credentials
3. Navigate to "New Article"
4. Attempt to drag and drop a file (e.g., image or .txt file) into the "Write your article" field
5. Attempt to paste a file or attachment into the same field

## Expected Result
The system should not accept file attachments.
