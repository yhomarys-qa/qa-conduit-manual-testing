# TC-056 - Updated username is displayed correctly after saving settings

## Steps
1. Log in to the application with valid credentials
2. Navigate to the Settings page
3. Update the "Username" field with a valid new value
4. Click on "Update Settings"
5. Observe the username displayed in the application header and profile section

## Expected Result
The updated username is displayed correctly in all relevant UI areas (e.g., top navigation bar and user profile page)
