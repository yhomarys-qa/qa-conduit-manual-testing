# TC-62 - User can log in using updated email after changing it in settings

## Steps
1. Log in to the application with valid credentials
2. Navigate to the Settings page
3. Update the "Email" field with a valid new email address
4. Click on "Update Settings"
5. Log out from the application
6. Attempt to log in using the updated email and the same password

## Expected Result
The user is able to log in successfully using the new email address
