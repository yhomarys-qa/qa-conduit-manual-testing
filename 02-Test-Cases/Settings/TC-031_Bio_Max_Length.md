# TC-031 - Bio field enforces a maximum length of 9000 characters

## Steps
1. Log in to the application with valid credentials
2. Navigate to the Settings page
3. Enter a text with more than 9000 characters in the "Bio" field (e.g., 9001 characters)
4. Click on "Update Settings"

## Expected Result
A validation error message is displayed indicating that the maximum allowed length of 9000 characters has been exceeded
