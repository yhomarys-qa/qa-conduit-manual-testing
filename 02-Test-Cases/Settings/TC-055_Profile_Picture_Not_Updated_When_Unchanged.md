# TC-055 - Profile picture is not updated when the field value is not modified

## Steps
1. Log in to the application with valid credentials
2. Navigate to the Settings page
3. Observe the current profile picture displayed in the UI
4. Do not modify the "Profile Picture URL" field
5. Click on "Update Settings"
6. Refresh the page or re-open the Settings page

## Expected Result
The profile picture remains unchanged after saving settings
