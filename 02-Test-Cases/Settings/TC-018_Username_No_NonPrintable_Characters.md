# TC-018 - Username does not allow non-printable or invalid whitespace characters

## Steps
1. Log in to the application with valid credentials
2. Navigate to the Settings page
3. Enter a username containing non-printable or invalid characters (e.g., leading/trailing spaces or hidden characters)
4. Click on "Update Settings"

## Expected Result
A validation error message is displayed indicating that invalid or non-printable characters are not allowed
