# TC-24 - Email must be unique in the system

## Steps
1. Log in to the application with valid credentials
2. Ensure an existing email is already registered in the system (e.g., "existing@email.com")
3. Navigate to the Settings page
4. Enter an email that is already registered in another account
5. Click on "Update Settings"

## Expected Result
A validation error message is displayed indicating that the email is already in use
