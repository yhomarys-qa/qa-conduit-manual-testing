# TC-42 - New password must not exceed 30 characters

## Steps
1. Log in to the application with valid credentials
2. Navigate to the Settings page
3. Enter a password with more than 30 characters in the "New Password" field (e.g., 31+ characters)
4. Click on "Update Settings"

## Expected Result
A validation error message is displayed indicating that the password must not exceed 30 characters
