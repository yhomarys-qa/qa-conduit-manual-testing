# TC-052 - User settings changes persist after successful update

## Steps
1. Log in to the application with valid credentials
2. Navigate to the Settings page
3. Update user data with valid values (e.g., Username, Email, Bio)
4. Click on "Update Settings"
5. Refresh the page or log out and log back in
6. Navigate again to the Settings page

## Expected Result
All saved changes remain visible after page refresh and after re-login
