# TC-64 - User cannot log in using the old email after email update

## Steps
1. Log in to the application with valid credentials
2. Navigate to the Settings page
3. Update the "Email" field with a valid new email address
4. Click on "Update Settings"
5. Log out from the application
6. Attempt to log in using the old email and the same password

## Expected Result
Login is rejected when using the old email address
