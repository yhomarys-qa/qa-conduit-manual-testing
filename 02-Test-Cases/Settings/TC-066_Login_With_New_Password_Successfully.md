# TC-066 - User can log in using the new password after updating it

## Steps
1. Log in to the application with valid credentials
2. Navigate to the Settings page
3. Update the "New Password" field with a valid password that meets complexity rules
4. Click on "Update Settings"
5. Log out from the application
6. Attempt to log in using the same email and the new password

## Expected Result
The user is able to log in successfully using the new password
