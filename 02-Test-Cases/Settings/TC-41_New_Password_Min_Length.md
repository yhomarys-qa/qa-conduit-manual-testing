# TC-41 - New password must have a minimum of 8 characters

## Steps
1. Log in to the application with valid credentials
2. Navigate to the Settings page
3. Enter a password with fewer than 8 characters in the "New Password" field (e.g., "Abc12")
4. Click on "Update Settings"

## Expected Result
A validation error message is displayed indicating that the password must have at least 8 characters
