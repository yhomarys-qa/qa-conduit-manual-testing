# TC-065 - Appropriate error message is displayed when logging in with an outdated email

## Steps
1. Log in to the application with valid credentials
2. Navigate to the Settings page
3. Update the "Email" field with a valid new email address
4. Click on "Update Settings"
5. Log out from the application
6. Attempt to log in using the old email and the correct password

## Expected Result
A clear and generic authentication error message is displayed (e.g., "Invalid credentials")
