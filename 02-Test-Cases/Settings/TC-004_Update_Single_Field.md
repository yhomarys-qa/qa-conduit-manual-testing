# TC-004 - User can update a single profile field without affecting others

## Steps
1. Log in to the application with valid credentials
2. Navigate to the Settings page
3. Note the current values of multiple fields (e.g., Username, Email, Bio)
4. Update only one field (e.g., change Bio to "Updated bio content")
5. Click on "Update Settings"

## Expected Result
The updated field is saved successfully
