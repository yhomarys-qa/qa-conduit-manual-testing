# TC-053 - Updated profile picture is displayed after saving settings

## Steps
1. Log in to the application with valid credentials
2. Navigate to the Settings page
3. Enter a valid image URL in the "Profile Picture URL" field (e.g., https://example.com/image.jpg)
4. Click on "Update Settings"
5. Navigate to the user profile or header section where the avatar is displayed

## Expected Result
The updated profile picture is displayed correctly in all relevant UI locations (e.g., header avatar and user profile page)
