# TC-68 - Appropriate error message is displayed when logging in with an old password

## Steps
1. Log in to the application with valid credentials
2. Navigate to the Settings page
3. Update the "New Password" field with a valid password
4. Click on "Update Settings"
5. Log out from the application
6. Attempt to log in using the same email and the old password

## Expected Result
A generic authentication error message is displayed (e.g., "Invalid credentials")
