# 🧪 Conduit Web Application – Manual QA Testing Project

## 📌 Overview
This project demonstrates manual QA testing practices applied to the Conduit (RealWorld) web application.  
It was created as part of my QA portfolio to simulate real-world testing activities.

## 🎯 Objective
The goal is to demonstrate my practical skills in:

- Requirements analysis  
- Test case design  
- Test execution (Smoke & Regression)  
- Bug identification and reporting  
- Test organization following industry best practices

## 🎯 Scope

The testing covers the following main modules:

- 🔐 Authentication (Login / Logout)
- 📝 Article Creation
- 🏠 Home Page
- ⚙️ User Settings
